## 3. SPONSOR LIFECYCLE 8 STAGE

```
Stage 1: INQUIRY
  → Sponsor submit form trên /movement/sponsors
  → Auto-acknowledge email
  → Status: New Lead

Stage 2: QUALIFICATION
  → Sponsor Manager + Founder review trong 3 ngày
  → Check fit, budget, ethics (no conflicting industries)
  → Status: Qualified hoặc Disqualified

Stage 3: PROPOSAL
  → Send Sponsor Proposal PDF (template per tier)
  → Schedule 30-60 phút call
  → Status: Proposal Sent

Stage 4: NEGOTIATION
  → Adjust scope, deliverable, timeline
  → Custom terms nếu cần
  → Status: In Negotiation

Stage 5: KYC + AGREEMENT
  → Send Sponsor Agreement
  → KYC: company registration + UBO + AML check
  → Sign (DocuSign or equivalent)
  → Status: Agreement Signed

Stage 6: PAYMENT
  → Invoice sent
  → Wire transfer hoặc Stripe Lane B
  → Confirm payment
  → Status: Paid

Stage 7: ACTIVATION + DELIVERY
  → Activate benefits per tier
  → Onboard call
  → Deliverable execution per timeline
  → Status: Active

Stage 8: REPORTING + RENEWAL
  → Quarterly impact report
  → Annual sponsor review
  → Renewal proposal at T-2 months
  → Status: Renewed | Lapsed | Upgraded
```

## 4. SPONSOR AGREEMENT TEMPLATE

### 4.1 Sections bắt buộc

1. **Parties** — DSTS legal entity (Lane B 501c3 hoặc VN Quỹ) + Sponsor
2. **Scope** — Tên gói + amount + duration
3. **Deliverables** — Liệt kê benefit per tier
4. **Timeline** — Milestones
5. **Payment terms** — Wire details, due date, taxes
6. **IP & Branding** — Logo usage, co-branding rules, approval workflow
7. **Confidentiality** — Mutual NDA cho info nhạy cảm
8. **Manifesto compliance** — Sponsor không request DSTS hứa kết quả (per Manifesto rule 2)
9. **Termination** — Conditions + refund clauses
10. **Force majeure**
11. **Dispute resolution** — Jurisdiction (VN hoặc US tùy lane)
12. **Anti-corruption clause** — FCPA + Vietnam Anti-Corruption Law 2018
13. **Data protection** — GDPR + NĐ 13/2023 compliance
14. **Signature block**

### 4.2 Quy tắc

- ❌ KHÔNG hứa media coverage cụ thể (vd "sẽ lên VTV1") — chỉ commit best effort
- ❌ KHÔNG hứa exclusivity industry-wide trừ Tier 6+ với điều kiện rõ
- ❌ KHÔNG cho phép sponsor chi phối nội dung không thuộc scope
- ✅ Brand approval workflow: sponsor có 5 ngày review co-branded content
- ✅ DSTS giữ creative final say cho mọi content
- ✅ Refund policy: prorate nếu DSTS cancel toàn bộ chương trình, KHÔNG refund nếu sponsor cancel sau Stage 6

## 5. KYC + AML CHECKLIST

### 5.1 KYC level theo tier

| Tier | KYC level |
|---|---|
| #1-3 (≤ $5K) | Basic: email + company registration check |
| #4-5 ($10-25K) | Standard: + UBO (Ultimate Beneficial Owner) declaration |
| #6-7 ($50-100K) | Enhanced: + officer ID verify + AML OFAC list check |
| #8-13 (≥ $25K with longer term) | Enhanced + Source of Funds declaration |
| #10, #13 (≥ $500K) | Full Due Diligence: site visit / video verification + legal opinion |

### 5.2 AML checks

- OFAC SDN list (US sanctions) — free via API
- EU Consolidated list — free
- UN Security Council list — free
- Politically Exposed Person (PEP) screening (Refinitiv / opensanctions.org)
- Adverse media check (Google + LexisNexis manual)

### 5.3 Red flag → escalate

- Country high-risk per FATF list (Iran, North Korea, etc.) → reject
- UBO không xác định được → request clarification, không xác định → reject
- Source of funds không nhất quán với industry → request bank statement
- Đề nghị thanh toán cash hoặc cryptocurrency → reject (compliance)
- Đề nghị deliverable ngoài scope (vd "tôi muốn quote anh Founder" về politics) → reject

## 6. FULFILLMENT TRACKER SCHEMA

### 6.1 Bảng `sponsors` (D1)

```sql
CREATE TABLE sponsors (
  id TEXT PRIMARY KEY,              -- DSTS-SPO-2026-NNNNN
  tier TEXT NOT NULL,               -- star-friend | star-supporter | ... | legacy-anchor
  company_name TEXT,
  contact_name TEXT NOT NULL,
  contact_email TEXT NOT NULL,
  contact_phone TEXT,
  amount_usd INTEGER NOT NULL,
  currency TEXT DEFAULT 'USD',
  status TEXT NOT NULL,             -- inquiry | qualified | proposal | nego | signed | paid | active | renewed | lapsed
  stage_entered_at INTEGER NOT NULL, -- timestamp last stage change
  kyc_level TEXT,                   -- basic | standard | enhanced | full
  kyc_completed_at INTEGER,
  agreement_url TEXT,               -- DocuSign URL
  agreement_signed_at INTEGER,
  payment_received_at INTEGER,
  activation_at INTEGER,
  renewal_due_at INTEGER,
  notes TEXT,                       -- JSON of internal notes
  created_at INTEGER DEFAULT (unixepoch()),
  updated_at INTEGER DEFAULT (unixepoch())
);
```

### 6.2 Bảng `sponsor_deliverables`

```sql
CREATE TABLE sponsor_deliverables (
  id TEXT PRIMARY KEY,
  sponsor_id TEXT NOT NULL,
  deliverable_type TEXT NOT NULL,   -- logo-placement | event-ticket | co-content | impact-report
  description TEXT,
  due_date INTEGER,
  status TEXT NOT NULL,             -- pending | in-progress | delivered | overdue
  delivered_at INTEGER,
  evidence_url TEXT,                -- proof: screenshot, photo, report PDF
  FOREIGN KEY (sponsor_id) REFERENCES sponsors(id)
);
```

### 6.3 Bảng `sponsor_communications`

```sql
CREATE TABLE sponsor_communications (
  id TEXT PRIMARY KEY,
  sponsor_id TEXT NOT NULL,
  type TEXT NOT NULL,               -- email | call | meeting | report
  direction TEXT NOT NULL,          -- inbound | outbound
  subject TEXT,
  body TEXT,
  attachment_url TEXT,
  scheduled_at INTEGER,
  completed_at INTEGER,
  contact_owner TEXT,               -- internal staff
  FOREIGN KEY (sponsor_id) REFERENCES sponsors(id)
);
```

## 8. CONFLICT + TERMINATION

### 8.1 Conflict resolution

Nếu sponsor có yêu cầu vi phạm Manifesto:
- Sponsor Manager refuse + cite Manifesto rule
- Escalate Founder nếu sponsor không accept
- Mediation: 30 ngày
- Nếu không resolve → terminate (per agreement clauses)

### 8.2 Termination clauses

DSTS có quyền terminate sponsor agreement nếu:
- Sponsor vi phạm anti-corruption clause
- Sponsor vi phạm Manifesto (hứa kết quả thay DSTS)
- Sponsor revealed liên quan industry conflict (vd: tobacco cho NDNUM trẻ em)
- Sponsor bị OFAC sanction hoặc adverse media nghiêm trọng
- Sponsor không pay > 60 ngày overdue

Termination process:
- 30 days written notice (trừ trường hợp khẩn cấp)
- Pro-rate refund cho deliverable chưa execute
- Public statement nếu cần (theo legal advice)

### 8.3 Audit + dispute

Sponsor có quyền:
- Request audit 1 lần / năm về expense liên quan sponsorship của họ
- Dispute deliverable không match agreement
- Withdraw nếu DSTS vi phạm material terms
